Streamable HTTP 版 v1.8.8 SSE 版（廃止予定）
バージョン: v2.0.1  |  発行日: 2026-02-28  |  対象プロトコル: Streamable HTTP (MCP 2025-03-26)
⚠️ プロトコル変更のお知らせ（重要）

旧バージョン（v1.8.8）で使用していた SSE（Server-Sent Events） は、 Claude（Anthropic 社）が「今後数カ月以内に廃止する」と正式に発表したプロトコルです。

代わりに Streamable HTTP が MCP サーバーの必須プロトコルとして義務付けられました。 今後 SSE のみに対応したサーバーは、Claude Desktop から接続できなくなります。

AiBridgeMCP v2 は Streamable HTTP に完全対応しています。 コネクタも新しい aibridge-connector-v2.exe をご使用ください。

この度は AiBridgeMCP をご利用いただきありがとうございます。
このツールは、Claude Desktop からあなたの ESP32 デバイスを操作するための「架け橋」となるプログラムです。

ℹ️ 旧版の aibridge-connector.exe（v1.8.8 SSE 用）とは別ファイルです。 必ず v2 のファイルをご使用ください。
🚀 セットアップ手順（3ステップ）
1. フォルダの準備

ダウンロードした connector フォルダを、パソコンの分かりやすい場所に移動してください。

おすすめの場所: C:\AiBridgeMCP
※ デスクトップやダウンロードフォルダでも動きますが、一度置いたら場所を変えないようにしてください。
2. ESP32 の STA IP アドレスを確認

あなたの AiBridgeMCP（ESP32）が接続されている Wi-Fi の STA IP アドレスをメモしてください。

例: 192.168.3.154
3. Claude Desktop の設定を書き換える

ここが一番のポイントです。慎重に行いましょう！

  1. Claude Desktop を完全に終了してください。
    ※ 右上の ✕ を押すだけでは完全に終了しない場合があります。以下の手順でタスクマネージャーから終了させてください。
    🗂 タスクマネージャーでの終了手順
    ① キーボードの Ctrl + Shift + Esc を同時に押す（タスクマネージャーが開きます）
    ② 「プロセス」タブの一覧から Claude または Claude Desktop を探す
    ③ 右クリック → 「タスクの終了」をクリック
    ④ タスクマネージャーを閉じる
  2. 設定ファイルのフォルダを開きます。
    以下の手順が最も簡単です：
    📂 フォルダの開き方
    ① キーボードの Windows キー（田）を押しながら R キーを押す
       →「ファイル名を指定して実行」という小さなウィンドウが開きます
    ② 入力欄に以下をそのままコピー＆ペーストして Enter キーを押す
    %APPDATA%\Roaming\Claude ③ Claude の設定フォルダが自動で開きます
  3. 開いたフォルダの中にある claude_desktop_config.json をメモ帳などで開きます。
  4. 中身を以下の内容に書き換えて保存してください。
{
  "mcpServers": {
    "AiBridgeMCP": {
      "command": "C:\\AiBridgeMCP\\connector\\aibridge-connector-v2.exe",
      "args": [
        "あなたのESP32のIPアドレス"
      ]
    }
  }
}
⚠️ 注意点
・パスの書き方: \（バックスラッシュ）は必ず \\ と2つ重ねて書いてください。
・IP アドレス: "192.168.3.154" のように、メモした IP アドレスを書き込んでください。
・ファイル名が aibridge-connector-v2.exe であることを確認してください。
✅ 動いているか確認する
  1. Claude Desktop を起動します。
  2. チャット入力欄の左下にある「＋」マークをクリックします。
  3. メニューに「コネクタ >」と表示されるのでクリックします。
  4. AiBridgeMCP がスライドスイッチ（🔵 青色・ON）で表示されていれば接続成功です！
  5. 念のため、Claude に話しかけて動作を確認してください。
    「ESP32 の状態を教えて」
    ESP32 の情報（IP アドレス、CPU 速度、空きメモリなど）が返ってくれば完璧です。
🛠 うまくいかないときは？
  • 「＋」→「コネクタ」に AiBridgeMCP が表示されない: claude_desktop_config.json の書き間違い（カンマやカッコの不足）がないか確認してください。Claude Desktop を再起動することも有効です。
  • ファイル名の確認: aibridge-connector-v2.exe（-v2 が付いているか）を確認してください。旧版（aibridge-connector.exe）は v2 では動作しません。
  • 通信エラーが出る: ESP32 と PC が同じ Wi-Fi につながっているか、IP アドレスが正しいか確認してください。
