🔭 AiBridgeMCP Quick Setup Guide

Streamable HTTP Edition v1.8.8 SSE Edition (deprecated)
Version: v2.0.1  |  Date: 2026-02-28  |  Protocol: Streamable HTTP (MCP 2025-03-26)
⚠️ Important: Protocol Change Notice

The SSE (Server-Sent Events) protocol used in the previous version (v1.8.8) has been officially announced for deprecation by Anthropic (Claude) within the coming months.

Streamable HTTP is now the required protocol for all MCP servers. Once SSE support is removed, MCP servers that rely solely on SSE will no longer be able to connect to Claude Desktop.

AiBridgeMCP v2 fully supports Streamable HTTP. Please use the new connector: aibridge-connector-v2.exe.

Thank you for using AiBridgeMCP.
This tool acts as a bridge that allows Claude Desktop to communicate directly with your ESP32 device.

📦 What's Included
  • connector folder
    • aibridge-connector-v2.exe — connection proxy program (v2, new)
    • This guide (PDF)
ℹ️ This is a different file from the old aibridge-connector.exe (used for v1.8.8 SSE). Always use the v2 file.
🚀 Setup Steps (3 Steps)
1. Place the Folder

Move the downloaded connector folder to a convenient location on your PC.

Recommended location: C:\AiBridgeMCP
※ The Desktop or Downloads folder also works, but do not move the folder after placing it — the path in the config must stay the same.
2. Find Your ESP32's STA IP Address

Note the STA IP address assigned to your AiBridgeMCP (ESP32) on your Wi-Fi network.

Example: 192.168.3.154
3. Edit the Claude Desktop Configuration

This is the most important step — please follow carefully.

  1. Fully quit Claude Desktop.
    Clicking the ✕ button alone may not close it completely. Use Task Manager to force-quit:
    🗂 How to Force Quit via Task Manager
    ① Press Ctrl + Shift + Esc simultaneously — Task Manager will open.
    ② Find Claude or Claude Desktop in the "Processes" list.
    ③ Right-click it → click "End Task".
    ④ Close Task Manager.
  2. Open the Claude Desktop configuration folder:
    📂 How to Open the Folder
    ① Press Windows key (⊞) + R at the same time.
       → A small "Run" dialog box will appear.
    ② Copy and paste the following into the input field, then press Enter:
    %APPDATA%\Roaming\Claude ③ The Claude configuration folder will open automatically.
  3. Open claude_desktop_config.json in the folder using Notepad or any text editor.
  4. Replace the entire contents with the following and save:
{
  "mcpServers": {
    "AiBridgeMCP": {
      "command": "C:\\AiBridgeMCP\\connector\\aibridge-connector-v2.exe",
      "args": [
        "your.ESP32.IP.address"
      ]
    }
  }
}
⚠️ Important Notes
· Backslashes: Each \ in the path must be written as \\ (doubled).
· IP address: Replace "your.ESP32.IP.address" with the actual IP, e.g. "192.168.3.154".
· Make sure the filename contains -v2: aibridge-connector-v2.exe.
✅ Verify the Connection
  1. Launch Claude Desktop.
  2. Click the "+" icon at the bottom-left of the chat input area.
  3. Select "Connectors >" from the menu.
  4. If AiBridgeMCP appears with a 🔵 blue toggle switch (ON), the connection is successful!
  5. To confirm it's working, ask Claude:
    "Tell me the status of the ESP32."
    If Claude responds with ESP32 details (IP address, CPU speed, free memory, etc.), everything is working perfectly.
🛠 Troubleshooting
  • AiBridgeMCP does not appear under "+" → "Connectors": Check claude_desktop_config.json for typos (missing commas or brackets). Try restarting Claude Desktop.
  • Wrong file: Confirm the filename is aibridge-connector-v2.exe (with -v2). The old aibridge-connector.exe is not compatible with v2.
  • Communication error: Make sure your ESP32 and PC are connected to the same Wi-Fi network and that the IP address is correct.
